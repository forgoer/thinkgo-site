The website for Thinkgo.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/thinkoner/thinkgo-site.git
$ cd thinkgo-site
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
