The website for Thinkgo.

## Getting started

Install dependencies:

``` bash
$ git clone https://github.com/thinkoner/thinkoner.github.io.git
$ cd thinkoner.github.io
$ npm install
```

Generate:

``` bash
$ hexo generate
```

Run server:

``` bash
$ hexo server
```
